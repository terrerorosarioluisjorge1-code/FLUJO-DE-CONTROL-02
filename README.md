# FLUJO-DE-CONTROL-02
##Resumen 
ESTE PROGRAMA FUE CREADO EN C# Y PERMITE EVLUAR EL RESULTADO FINAL DE CALIFICACIONES DE ESTUDIANTES YO LE PUSE PARA QUE PUEDA MEDIR LAS CALIFICACIONES DE 2-3-4 ESTUDIANTES UTILIZANDO BUCLES FOR WHILE 

EL PROGRAMA SOLICITA EL NUMERO DE ESTUDIANTES QUE DESEE CALCULAR SU CALIFICACION SU RESULTADO SE DETERMINARA DEPENDIENDO DEL NUMERO DE FINAL DE CALIFICACIONES SUMADAS SI TERMNA POR DEBAJO DE 70 ESTA REPROBADO FALSE   Y SI TERMINA POR ENCIMA DE 70 ES TRUE VERDADERO QUE SERIA APROBADO
